## Component Library

### Buttons
- **Primary:** Indigo background, white text, full width
- **Secondary:** Indigo outline, indigo text, full width
- **Text:** No background, gray text, centered
- **Icon:** Square, icon only, transparent

### Cards
- **Topic Card:** White background, shadow, rounded 12px, padding 16px
- **Stats Card:** Light gray background, rounded 16px, padding 20px
- **Empty State:** Center aligned, icon + text + button

### Inputs
- **Text Field:** Outlined, gray border, rounded 8px, 16px padding
- **Text Area:** Same as text field, multi-line, min height 200px
- **Switch:** Material switch, indigo when on
- **Checkbox:** Material checkbox, indigo when checked

### Status Badges
- **Overdue:** Red background, white text, rounded pill
- **Due Today:** Orange background, white text, rounded pill
- **Upcoming:** Green background, white text, rounded pill
